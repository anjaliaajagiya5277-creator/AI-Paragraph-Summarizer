# 🤖 AI Paragraph Summarizer

An AI-powered web application that summarizes long paragraphs into short and meaningful summaries using Google's Gemini AI.

---

## Features

- Summarize long paragraphs instantly
- Clean and modern user interface
- Built with Flask
- Powered by Gemini AI
- Copy summary with one click
- Error handling
- Responsive Design

---

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Google Gemini API
- python-dotenv

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Paragraph-Summarizer.git
```

Move into the project folder

```bash
cd AI-Paragraph-Summarizer
```

Create a .env file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```

---

## How It Works

1. Enter a paragraph.
2. Click **Summarize**.
3. Gemini AI processes the text.
4. A concise summary is displayed.
5. Copy the summary if needed.
